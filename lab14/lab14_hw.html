Who are some very powerful supers? Lets make a different word cloud with the superhero_powers data. Use the provided code to make the frequency table, then make a word cloud with it. You can use hero_names for the labels, and sum_powers for size.

power_quantity <- superhero_powers %>% 
  pivot_longer(-hero_names, names_to = "power", values_to = "yes_or_no") %>% 
  group_by(hero_names) %>% 
  mutate(sum_powers = sum(yes_or_no, na.rm=T)) %>%
  arrange(desc(sum_powers), hero_names, desc(yes_or_no))
